Quick Start

1Get a Free Finnhub API Key

Go to finnhub.io/register and create a free account. Your API key will be on the dashboard immediately after signup.

Free tier includes: 60 API calls/minute, real-time US stock data, WebSocket streaming, and market news.

2Open the Dashboard

Open dashboard.html in any modern web browser (Chrome, Firefox, Safari, Edge).

3Enter Your Settings

On the setup screen, enter your Finnhub API key, starting account balance, and max risk percentage per trade.

Click "Launch Dashboard" and you're live!

4Demo Mode (No API Key)

Leave the API key blank and click Launch to run in demo mode with simulated data. Great for exploring the dashboard before connecting live data.

How the Signal Engine Works

The signal engine scores each stock on a multi-factor technical analysis model:

RSI (Relative Strength Index) — Oversold (<30) = bullish, Overbought (>70) = bearish

MACD — Positive crossover = bullish, negative = bearish

Moving Averages — Price above SMA20/EMA9 = bullish

Bollinger Bands — At lower band = potential bounce, upper band = potential reversal

VWAP — Above = institutional buying pressure

Volume — High volume confirms the direction of the move

Each factor is weighted and combined into a composite score that generates BUY, SELL, or WATCH signals with confidence levels.

Tips for Success

Start small. Never risk more than 1-2% of your account on a single trade.

Always use stop losses. No exceptions. Ever.

Aim for 2:1 risk/reward minimum. The position calculator helps you stay disciplined.

Trade the first 2 hours and last hour. That's where the volume and volatility are.

Keep a trade journal. Use the built-in journal to review what's working and what isn't.

Never revenge trade. If you hit 3 losses in a day, stop. Come back tomorrow.
